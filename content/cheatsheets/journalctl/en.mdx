## Service-scoped tail

```bash
journalctl -u ssh -f
```

## Time-bounded ISO output

```bash
journalctl -u nginx --since "2025-02-18 00:00:00" --until "2025-02-18 01:00:00" -o short-iso
```

## Notes

- Prefer **unit-scoped** queries before global keyword greps.
- Build a hypothesis first; then filter aggressively.
