updatedAt: '2025-02-01'
---

## Pretty print

```bash
curl -sS https://api.example.com/v1/meta | jq '.'
```

## Select nested keys

```bash
curl -sS https://api.example.com/v1/meta | jq '.build.version'
```

## Notes

- `curl -i` first when you need headers; then refine JSON parsing.
